Encapsulation:-
=> Hiding the implementation details.
=>In which one thing is included with other thing so that inclusive things will not appear.

Polymorphism :-
=> A way to achieve a single action in different ways.

Inheritance :- 
=> parent-child relationship.In which a class takes all the characteristics of parent class
=> child => Inherits => parent
=> example :- gene,dna
